# 🚀 Meus Primeiros Passos em JavaScript

Este repositório foi criado para documentar a minha evolução na lógica de programação utilizando **JavaScript**. 

## 📌 Sobre o Projeto
O primeiro script incluído aqui (`progresso.js`) é um contador lógico que calcula a minha jornada acadêmica no curso de **Análise e Desenvolvimento de Sistemas (ADS)** no **UniCEUB**.

### O que este código faz:
- Utiliza variáveis para armazenar o progresso do curso.
- Realiza cálculos matemáticos para definir a porcentagem de conclusão.
- Utiliza estruturas condicionais (`if/else`) para exibir mensagens motivacionais baseadas no progresso.

## 🛠️ Tecnologias Utilizadas
- **JavaScript** (Lógica de programação)
- **Git & GitHub** (Versionamento e portfólio)
